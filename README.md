# Ke Liu Homepage

This project is a local static academic homepage for 刘珂（Ke Liu）, Associate Professor at the Institute of Computing Technology, Chinese Academy of Sciences.

## Structure

- `index.html` - regenerated homepage entry.
- `assets/css/style.css` - CSS inherited from the reference academic homepage layout.
- `assets/images/` - downloaded image resources, including the local profile photo and ICT/UCAS logo.
- `assets/source/` - fetched source pages kept as references for the profile, students, publications, and patents.

Open `index.html` directly in a browser to view the page.
